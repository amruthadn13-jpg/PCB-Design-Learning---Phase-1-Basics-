# PCB Design Learning - Phase 1 (Basics)

## Overview

This repository documents the progress of learning PCB design starting from the fundamentals. Phase 1 focuses on understanding the basic concepts of PCB design and getting familiar with the ECAD tool workflow.

---

## Phase 1 Goal

The objective of this phase is to build a strong foundation in PCB design by understanding core concepts and practicing schematic creation using a design tool.

---

## Concepts Covered

* Printed Circuit Board (PCB) and Printed Circuit Board Assembly (PCBA)
* Difference between schematic design and PCB layout
* Basic electronic components:

  * Resistors
  * Capacitors
  * Integrated Circuits
  * Connectors
* Electrical connections:

  * Nets
  * Traces
  * Vias

---

## Tool Learning (KiCad)

The following operations were practiced in the schematic editor:

* Creating a new project
* Adding components using shortcut key (A)
* Wiring components using shortcut key (W)
* Understanding basic schematic flow

---

## Practice Project

### LED Circuit Design

A simple LED circuit was designed to understand basic connectivity and component placement.

#### Components Used:

* LED (Red)
* Resistor
* Power Source (Battery)

#### Description:

The circuit connects a power source to an LED through a resistor. The resistor limits the current to ensure safe operation of the LED. This basic design helps in understanding how components are connected in a schematic before converting it into a PCB layout.
